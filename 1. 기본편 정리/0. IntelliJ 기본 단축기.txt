1. IntelliJ 기본 단축키
   - Alt + 1 : 프로젝트 창 끄고 키고 포커스 위치.
   - Space : 프로젝트 창에 포커스가 위치한 에디터 창 미리보기
   - Ctrl + Shift + F12 : 에디터 창만 나타내기
   - Ctrl + Tab : 에디터 창 이동
   - Ctrl + Alt + Insert(에디터), Alt + Insert(프로젝트 창) : 새 파일 생성
   - Ctrl + <- -> : 단어별 커서 이동
   - Home, End : 라인 시작/끝 커서 이동
   - PgUp, PgDn : 페이지 위/아래 커서 이동
   - Ctrl + W(확장), Ctrl + Shift + W(축소) : 코드의 영역을 확장하고 축소할 때 사용
   - Ctrl + /(한 줄 주석), Ctrl + Shift + /(블록 주석) : 주석처리
   - Tap(인덴트), Shift + Tab(인덴트 취소) : 들여쓰기/들여쓰기 취소
   - Ctrl + Alt + L : 코드 전체를 자동 인덴트 해줌
   - Ctrl + E : 최근 열었던 파일 확인 열기
   - Alt + Enter : 코드에서 오류가 발견된 경우 수정 제안 목록 표시
   - Shift + Alt + Enter : 수정 제안 목록의 첫 번째 제안 적용
   - F2 : 다음 오류, 경고 또는 제안으로 점프할 수 있음
   - Esc : 에디터에 포커스 위치
   - Alt + F7 : 어떤 항목이 사용된 위치를 검색, 이 인터페이스가 사용된 모든 곳이 검색 창에 표시.
   - Shift + Ctrl + Enter : 현재 구문 완성
   - Shift + Ctrl + Alt + T : 사용 가능한 리팩토링 옵션이 표시되고  
   - Alt + Insert(에디터) : 코드 삽입(Constructor, get/setter, toString 등)
   - Ctrl + D : 한줄 복사하고 바로 붙여넣기
